Laboratorio 2

JavaScript es un lenguaje de programación interpretado que se utiliza principalmente en el desarrollo web para crear páginas interactivas y dinámicas. Originalmente fue diseñado para ejecutarse en navegadores web y permitir la manipulación de elementos HTML y CSS. Hoy en día, JavaScript es también utilizado en servidores (gracias a tecnologías como Node.js), en aplicaciones móviles, de escritorio y en el desarrollo de videojuegos.

Semejanzas entre Javascript y Java

  • Sintaxis similar: Ambos lenguajes tienen una sintaxis basada en C, lo que significa que comparten estructuras de control (como if, for, y while) y operadores comunes.
  • Orientación a objetos: Ambos lenguajes soportan la programación orientada a objetos, aunque de manera diferente. Java utiliza clases, mientras que JavaScript utiliza prototipos.
  • Nombre: Aunque sus nombres son similares, esto es solo una coincidencia. JavaScript fue nombrado así en parte para aprovechar la popularidad de Java en ese momento.

Diferencias clave

  • Propósito y Uso: Java es un lenguaje de programación de propósito general que se utiliza para una amplia gama de aplicaciones, desde aplicaciones móviles (Android) hasta sistemas empresariales complejos. JavaScript se centra en la web y en la interactividad del usuario en los navegadores, aunque también se usa en otros entornos.
  • Compilación vs Interpretación: Java es un lenguaje compilado que se convierte en bytecode ejecutado en la JVM (Java Virtual Machine). JavaScript es interpretado directamente en el navegador o en un entorno como Node.js.
  • Tipos de Programación: Java es fuertemente tipado y requiere la declaración de tipos de variables, mientras que JavaScript es débilmente tipado y permite una mayor flexibilidad en la asignación de tipos a las variables.

En resumen, aunque Java y JavaScript comparten una sintaxis similar y algunas características de la programación orientada a objetos, son lenguajes diferentes diseñados para propósitos distintos.
